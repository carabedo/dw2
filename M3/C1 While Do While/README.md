## While

```js
while(true){
//codigo
}
```
Podemos usar cualquier expresion que devuelva un booleano.

```js
while( 10 < 3){
//codigo
}
```

Como lo usamos? Como hacemos que se detenga?

```js
let i = 0
while( i < 5){
console.log('Hola!')
i= i + 1
}
```
